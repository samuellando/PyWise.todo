Tasks you enter should be in the form:
+XX @DD/MM/YY description.

where:
+ defines the project name
@DD/MM/YYYY defines a date

While the tag for the Project name can be what ever you want, the date tag must be 
the the format of @DD/MM/YYYY for the task to properly get formated by the 
Sort by Date function.

Type h in the comand box to show the list of all avalabe comands.
